8051ELL库-片内外设篇-可编程计数器

8051MCU
8051 ELL函数库指南
发布日期:   2021-03-01

STC8系列

外设介绍

STC8内部集成了可编程计数器阵列模块，可用于软件定时器、外部脉冲捕获、高速脉冲输出和PWM脉宽调制输出。STC8A系列内部有四组，其他系列可能有所不同。

函数库提供了可编程计数器的四种应用方案，可以用于扩展定时器、PWM、高速脉冲输出、输入捕获。


API手册

1. PCA计数器初始化函数

函数名 描述
FSCSTATE PCA_CNT_Init(PCACLKSrc_Type clkSrc,FUNSTATE run) PCA计数器初始化

SPIClkSrc_Type 枚举体说明
typedef enum
{
  PCA_SCLK_DIV_12 = 0x00,  // 系统时钟12分频做时钟源
  PCA_SCLK_DIV_2  = 0x02,  // 系统时钟2分频做时钟源
  PCA_TIMER0      = 0x04,  // 定时器0溢出脉冲做时钟源
  PCA_ECI         = 0x06,  // ECI引脚外部输入时钟做时钟源
  PCA_SCLK_DIV_1  = 0x08,  // 系统时钟1分频做时钟源
  PCA_SCLK_DIV_4  = 0x0A,  // 系统时钟4分频做时钟源
  PCA_SCLK_DIV_6  = 0x0C,  // 系统时钟6分频做时钟源
  PCA_SCLK_DIV_8  = 0x0E   // 系统时钟8分频做时钟源
}  PCACLKSrc_Type;

2. PCA作为PWM操作函数

函数名 描述
FSCSTATE PCA0_PWM_Init(PCA_PWMBits_Type pwmBits,uint16_t duty) PCA0作为PWM初始化
FSCSTATE PCA1_PWM_Init(PCA_PWMBits_Type pwmBits,uint16_t duty) PCA1作为PWM初始化
FSCSTATE PCA2_PWM_Init(PCA_PWMBits_Type pwmBits,uint16_t duty) PCA2作为PWM初始化
FSCSTATE PCA3_PWM_Init(PCA_PWMBits_Type pwmBits,uint16_t duty) PCA3作为PWM初始化

PCA_PWMBits_Type 枚举体说明
typedef enum
{
	PCA_PWM_8Bit  = 0x00,   // 8位分辨率PWM模式 
	PCA_PWM_7Bit  = 0x01,	// 7位分辨率PWM模式
	PCA_PWM_6Bit  = 0x02,	// 6位分辨率PWM模式
	PCA_PWM_10Bit = 0x03	// 10位分辨率PWM模式
}	PCA_PWMBits_Type;

函数名 描述
FSCSTATE PCAx_PWM_6BITS_CTRL(duty) PCAx作为6位PWM初始化和控制
FSCSTATE PCAx_PWM_7BITS_CTRL(duty) PCAx作为7位PWM初始化和控制
FSCSTATE PCAx_PWM_8BITS_CTRL(duty) PCAx作为8位PWM初始化和控制
FSCSTATE PCAx_PWM_10BITS_CTRL(duty) PCAx作为10位PWM初始化和控制

Duty： 占空比计算说明
/**
  * 说明： 1. xxH：H为十六进制表示，C语言代码里用0x表示;
  *       2.CLKSrc：为PCA_CNT时钟源,单位Hz;
***/

6位分辨率模式：
占空比 = (40H-duty)/40H
频率  = CLKSrc/64

7位分辨率模式：
占空比 = (80H-duty)/80H
频率  = CLKSrc/128 

8位分辨率模式：
占空比 = (100H-duty)/100H
频率  = CLKSrc/256

10位分辨率模式：
占空比 = (400H-duty)/400H
频率  = CLKSrc/1024

代码演示

这里以PCA作为扩展定时器演示：

#include "8051_ELL_Lib.h"

void STC8x_PCA_Config(void)
{
    /* 先初始化PCA_TIM */
	PCA0_TIM_Init(1000);   //定时1s
	NVIC_PCA0_TIM_POP_Init(ENABLE); //启动中断
	
    /* 再初始化 PCA_CNT */
	PCA_CNT_Init(PCA_SCLK,ENABLE);  
	NVIC_PCA_CNT_Init(NVIC_PR1,DISABLE); //启动PCA计数器
}

void PCA_ISRQ_Handler(void)
{
    if(PCA0_GET_FLAG())
    {
        PCA0_CLEAR_FLAG();
        PCA0_TIM_POP_ReValue();  //计数器重装载
        P10 =!P10;
    }
}

int main(void)
{
    STC8x_PCA_Config();
    for(;;)
    {
        
    }
}

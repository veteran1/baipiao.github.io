8051ELL库-片内外设篇-比较器

8051MCU
8051 ELL函数库指南
发布日期:   2021-03-16

STC8系列

外设介绍

STC8系列MCU集成了一个比较器，可以用于电压比较、掉电检查以及电量检测等功能。

比较器有两级滤波：模拟滤波和数字滤波。模拟滤波可以过滤掉比较输入信号中的毛刺信号，数字滤波可以等待输入信号稳定以后再进行比较，比较结果可以通过寄存器获得，也可以直接输出到IO口。输出到IO口可用作外部事件的触发信号和反馈信号，扩大了比较器的应用范围。

比较器可以联动ADC的通道，来灵活扩展IO，通过分时的手段，可以实现比较器复用的功能，但是要注意的是，不同的型号ADC的通道引脚有所不同，不过ELL库已经适配了大部分，但是由于数量较多，为防止遗漏和错误，谨慎起见，还请开发者对照官方数据手操作。


API手册

1. 比较器初始化函数

函数名 描述
FSCSTATE COMP_Init(const COMP_InitType *COMPx) 比较器初始化

2. 比较器输出IO口切换函数

函数名 描述
FSCSTATE GPIO_COMP_SWPort(GPIOSWPort_Type Port) 比较器输出IO口切换

3. 比较器中断初始化函数

函数名 描述
FSCSTATE NVIC_COMP_Init(NVICPri_Type priority,COMPTri_Type triMode) 比较器中断初始化
FSCSTATE NVIC_COMP_CTRL(run) 比较器中断控制

COMP_InitType 结构体说明
typedef enum
{
  COMP_Tri_Null    = 0x00,  //不触发
  COMP_Tri_Falling = 0x10,  //下降沿触发
  COMP_Tri_Rising  = 0x20,  //上升沿触发
  COMP_Tri_Edge    = 0x30   //边沿触发
} COMPTri_Type;

3. 比较器中断服务函数

函数名 描述
void COMP_ISRQ_Handler(void) 比较器中断服务函数

4. 比较器中断操作函数

函数名 描述
NVIC_COMP_CTRL(run) 比较器中断控制
COMP_GET_FLAG() 比较器获取标志位
COMP_CLEAR_FLAG() 比较器清理标志位
COMP_GET_RES() 比较器获取结果

COMP_InitType 结构体说明
typedef struct
{
    uint8_t PEPin;            //比较器正极输入选择位
    uint8_t NEPin;	        //比较器负极输入选择位
    FUNSTATE AC_FTW;        //比较器0.1us模拟滤波 0 DISABLE / 1 ENABLE
    uint16_t DC_FTW;          //比较器数字滤波 0代表关闭 最大0x3F
    COMPDIRPut_Type DIRPut; //比较器输出方向选择
    FUNSTATE Output;        //比较器输出控制位
    FUNSTATE run;           //比较器运行控制位
}   COMP_InitType;

代码演示

```C

#include “8051_ELL_Lib.h”

void STC8x_COMP_Config(void)
{
COMP_InitType COMP_InitStruct={0};

GPIO_MODE_IN_FLOATING(GPIO_P3,Pin_7);   //浮空输入
COMP_InitStruct.PEPin = COMP_PE_P37;    //正极选择P37
COMP_InitStruct.NEPin = COMP_IN_REV;    //负极选择内部参考电压
COMP_InitStruct.AC_FTW = ENABLE;       //使能模拟滤波
COMP_InitStruct.DC_FTW = 0x08;        //开启数字滤波，滤波级别0x08
COMP_InitStruct.DIRPut = COMP_DIR_Forward;  // 正向输出
COMP_InitStruct.Output = ENABLE;     //输出比较结果
COMP_InitStruct.run = ENABLE;       //使能运行
COMP_Init(&COMP_InitStruct);
NVIC_COMP_Init(NVIC_PR0,COMP_Tri_Edge);  //开启中断

}

void COMP_ISRQ_Handler (void)
{
uint8_t res;
if(COMP_GET_FLAG())
{
COMP_CLEAR_FLAG();
res = COMP_GET_RES();
}
}

int main(void)
{
STC8x_COMP_Config();
for(;;)
{

}

}
